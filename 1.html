What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. Every website you visit is built on HTML.

How does a browser read HTML?

When you visit a URL, the browser sends a request to a server and receives an HTML file in return.

The browser reads through the HTML top to bottom, recognising opening and closing tags and building a tree structure called the DOM (Document Object Model).

Using the DOM tree, the browser calculates layout and paints pixels on your screen — producing the visual page you see.

Core Concepts

Elements & Tags

An HTML element consists of an opening tag, content, and a closing tag:

<p>This is a paragraph.</p>

Some elements are self-closing and need no closing tag:

<img src="photo.jpg" alt="A photo" />

Attributes

Attributes provide extra information about an element. They live inside the opening tag:

<a href="https://example.com" target="_blank">Visit</a>

Common attributes: id, class, src, href, alt, style

Page Structure

Every valid HTML page follows this structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Common HTML Tags

A searchable reference of the most frequently used HTML tags.

Tag Name Purpose Example
<h1> Heading 1 Main page title <h1>Hello</h1>
<p> Paragraph Blocks of text <p>Some text</p>
<a> Anchor Hyperlinks <a href="#">Click</a>
<img> Image Displays an image <img src="pic.jpg" alt="desc" />
<ul> Unordered List Bullet point list <ul><li>Item</li></ul>
<ol> Ordered List Numbered list <ol><li>Item</li></ol>
<table> Table Displays tabular data <table>...</table>
<form> Form Collects user input <form>...</form>
<div> Division Generic container block <div class="box">...</div>
<span> Span Inline container <span style="color:red">text</span>
