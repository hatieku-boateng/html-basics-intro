Introduction to HTML

Your first step into web development.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

Key Concepts

Elements & Tags

HTML is made up of elements represented by tags, like <p> for paragraphs and <h1> for headings.

Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important).

Attributes

Tags can have attributes that provide extra information, such as src for images or href for links.

Why Learn HTML?

Benefits

  • It is the foundation of every website.
  • Easy to learn and get started with.
  • Works alongside CSS and JavaScript.

Steps to Get Started

  1. Learn the basic HTML structure.
  2. Understand common tags and their purpose.
  3. Build your first webpage.
  4. Style it with CSS.

Common HTML Tags

Here is a quick reference table of frequently used HTML tags:

Tag Name Purpose Example
<h1> Heading 1 Main page title <h1>Hello</h1>
<p> Paragraph Blocks of text <p>Some text</p>
<a> Anchor Hyperlinks <a href="#">Click</a>
<img> Image Displays an image <img src="pic.jpg" alt="desc" />
<ul> Unordered List Bullet point list <ul><li>Item</li></ul>
<ol> Ordered List Numbered list <ol><li>Item</li></ol>
<table> Table Displays tabular data <table>...</table>
<form> Form Collects user input <form>...</form>
