Tips for the Playground

  • You can use any valid HTML tag — headings, images, tables, forms, links, and more.
  • External images need a full URL: src="https://..."
  • Use <br> for line breaks and <hr> for a horizontal rule.
  • Click Reset to restore the default code, or Clear to start fresh.
  • Right-click in the preview and choose "View Page Source" to inspect the rendered HTML.
